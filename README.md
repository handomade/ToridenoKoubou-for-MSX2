＠遊び方

＊タイトル画面　

一人用は１、二人で遊ぶなら２を押してください。

＊ゲーム

ang（角度）vel（強さ）を入力して、FIRE!ボタンで攻撃です。

基本的には角度はいじらないで強さを風の強さを見て調整するといいと思います。

入力欄はTABキーとリターンキーで次へ進みます。

FIREボタンがアクティブな時にリターンキーを押すと攻撃します。TABキーを押すとangに戻ります。

どちらかが3回攻撃を受けてしまうと負け。

リターンキーでタイトル画面に戻ります。
